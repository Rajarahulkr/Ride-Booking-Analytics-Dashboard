Ride Booking Analytics Dashboard

This Power BI project provides a comprehensive analysis of ride booking operations using a dataset containing 150,000 ride records.

Objectives
Analyze booking trends and ride performance.
Monitor revenue generation.
Identify cancellation patterns.
Evaluate customer and driver satisfaction.
Compare vehicle category performance.
Key Insights
93K rides successfully completed.
₹47.26M total booking value generated.
Average customer rating of 4.4/5.
Average driver rating of 4.23/5.
Detailed analysis of cancellations, incomplete rides, and operational bottlenecks.
